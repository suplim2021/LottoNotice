# ⚙️ เลขอั้น Generator (Lotto Notice)

เว็บแอปพลิเคชันรูปแบบ Single Page สำหรับสร้างและส่งออกภาพประกาศ "เลขอั้น" หรือ "เลขจ่ายครึ่ง" เพื่อใช้อัปเดตและแจ้งเตือนผ่านช่องทางต่างๆ (เช่น เว็บไซต์, LINE, Facebook) ได้อย่างสวยงาม รวดเร็ว และเป็นระเบียบ

## 🚀 ฟีเจอร์หลัก (Features)

- **สร้างภาพเลขอั้นได้เรียลไทม์:** แค่วางตัวเลขลงในแบบฟอร์ม ทางขวาจะแสดงผลการ์ดภาพที่จัดหน้าตัวเลขเป็นคอลัมน์สวยงามทันที
- **แบ่งหมวดหมู่ตัวเลขอัตโนมัติ:** 
  - เลข 3 ตัว (เลขติดกัน เช่น `359`) จะอยู่ในหมวด "บน-เต็ง-โต๊ด หกกลับ สามกลับ"
  - เลข 2 ตัว (เลขมีขีดคั่น เช่น `05-50`) จะอยู่ในหมวด "2 ตัว บน-ล่าง"
- **ปรับแต่งสี (Color Theme):** สามารถเลือกสีหลักของการ์ดได้อิสระ โดยระบบจะจำสีที่ตั้งไว้ใน LocalStorage สำหรับการใช้งานครั้งต่อไป
- **ตั้งค่าเกี่ยวกับเวลา:** กำหนดวันที่และเวลาปิดรับรายวันได้
- **โหมดสำหรับการพิมพ์ (Print Mode):** สามารถเปิดโหมด "โชว์เฉพาะตัวเลข + วันที่" เพื่อซ่อนข้อความส่วนเกินสำหรับพิมพ์ออกมาเช็คแบบรวดเร็ว
- **บันทึกเป็นรูปภาพ (Export JPG):** คอนเวิร์ตภาพจากการ์ด HTML ไปเป็นไฟล์ ​JPG ด้วยปุ่มเดียว
- **พิมพ์หรือเซฟเป็น PDF:** รองรับการสั่งพิมพ์ A4 ในสัดส่วนและการแสดงผลสีที่ถูกต้อง (Print-color-adjust)

## 🛠️ เทคโนโลยีที่ใช้ (Tech Stack)

- **HTML5 และ CSS3 (Vanilla)**
- **JavaScript (Vanilla JS):** สำหรับฟังก์ชันการทำงาน จัดโครงสร้างตัวเลข โหลด/จำค่าสี
- **Font:** ใช้ [Google Fonts](https://fonts.google.com/) (`Kanit` และ `Sarabun`)
- **[html2canvas](https://html2canvas.hertzen.com/):** External Library ที่ใช้เพื่อแคปเจอร์ DOM render ให้ออกมาเป็นรูปภาพ

## 📦 วิธีการเปิดใช้งาน (How to use)

ไม่จำเป็นต้องติดตั้งโปรแกรมหรือรันเซิร์ฟเวอร์ใด ๆ:

1. ดับเบิลคลิกเปิดไฟล์ `lotto.html` ด้วยเว็บเบราว์เซอร์ (แนะนำ Google Chrome, Edge หรือ Safari)
2. ที่ฝั่งซ้าย (แผงควบคุม) ให้กำหนด **วันที่** และ **เวลาปิดรับ**
3. พิมพ์ตัวเลขที่ต้องการลงในช่อง "ตัวเลข (บรรทัดละเลข)"
   - *ตัวอย่างเลข 3 ตัว:* `359` `123`
   - *ตัวอย่างเลข 2 ตัว:* `05-50` `11`
4. (ตัวเลือก) ปรับการตั้งค่าสี หรือเปิดโหมดพิมพ์แบบเรียบง่ายตามความต้องการ
5. เสร็จแล้วสามารถกดปุ่ม **📷 Export JPG** ระบบจะดาวน์โหลดรูปภาพให้ทันที หรือกด **🖨️ Print / Save PDF** ถ้าต้องการออกเครื่องพิมพ์

- `CLAUDE.md` - กฎหรือ Prompt Guidelines พิเศษสำหรับการแก้ไข/เขียนโค้ดด้วย AI
